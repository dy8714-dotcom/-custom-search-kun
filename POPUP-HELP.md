# ポップアップブロック解除ガイド

「カスタマイズ検索くん」の一括検索機能を使用する際、ブラウザのポップアップブロック機能により、複数のタブが開かない場合があります。

このガイドでは、各ブラウザでのポップアップブロック解除方法を説明します。

## 🔍 問題の症状

- 一括検索ボタンを押しても、最初の1～2個のタブしか開かない
- 「ポップアップがブロックされました」という警告が表示される
- アドレスバーに🚫マークが表示される

## ✅ 解決方法

### Google Chrome / Microsoft Edge

1. **一括検索ボタンをクリック**
2. アドレスバーの右側に表示される **🚫 ポップアップがブロックされました** アイコンをクリック
3. **「常に許可」** を選択
4. もう一度 **一括検索ボタン** をクリック

#### 設定から変更する場合:

1. Chrome設定 → プライバシーとセキュリティ → サイトの設定
2. ポップアップとリダイレクト
3. 「許可」セクションに本サイトのURLを追加

### Mozilla Firefox

1. **一括検索ボタンをクリック**
2. アドレスバーの左側に表示される **盾アイコン** をクリック
3. **「このサイトでポップアップブロッカーを無効にする」** を選択
4. ページを再読み込みして、もう一度 **一括検索ボタン** をクリック

#### 設定から変更する場合:

1. Firefox設定 → プライバシーとセキュリティ
2. 許可設定 → ポップアップウィンドウをブロックする → 例外サイト
3. 本サイトのURLを追加

### Safari (Mac)

1. **Safari環境設定** を開く（⌘ + ,）
2. **Webサイト** タブをクリック
3. 左側メニューから **ポップアップウインドウ** を選択
4. 本サイトを **「許可」** に設定

または:

1. 一括検索時に表示される確認ダイアログで **「許可」** をクリック

### その他のブラウザ

#### Brave

1. アドレスバー右側の **Braveシールドアイコン** をクリック
2. 「シールドが有効」を **無効** に変更
3. ページを再読み込み

#### Opera

1. アドレスバー左側の **錠前アイコン** をクリック
2. サイトの設定
3. ポップアップとリダイレクト → **許可**

#### Vivaldi

1. アドレスバー右側の **サイト情報アイコン** をクリック
2. サイトの設定
3. ポップアップとリダイレクト → **許可**

## 🔐 セキュリティについて

**Q: ポップアップを許可しても安全ですか？**

A: はい。このツールは以下の理由で安全です：

- ✅ 完全にクライアントサイド（ブラウザ内）で動作
- ✅ サーバーにデータを送信しない
- ✅ 悪意のある広告やマルウェアなし
- ✅ オープンソース（コードが確認可能）
- ✅ 検索サイトのURLは自分でカスタマイズ可能

ポップアップは、あなたが入力した検索キーワードで各検索サイトを開くためにのみ使用されます。

## 💡 ヒント

### 一度設定すれば永続的に有効

ポップアップブロックを一度解除すれば、次回からは自動的にすべてのタブが開きます。毎回設定する必要はありません。

### テスト方法

1. 2つの検索窓にキーワードを入力
2. 検索サイトを1つ選択
3. 一括検索ボタンをクリック
4. 2つのタブが開けば成功！

### それでも開かない場合

1. ブラウザを最新バージョンに更新
2. 拡張機能（広告ブロッカーなど）を一時的に無効化
3. シークレット/プライベートモードで試す
4. 別のブラウザで試す

## 📞 サポート

問題が解決しない場合は、以下の情報をご確認ください：

- 使用しているブラウザ名とバージョン
- エラーメッセージ（あれば）
- 開発者ツールのコンソールログ（F12キーで開く）
